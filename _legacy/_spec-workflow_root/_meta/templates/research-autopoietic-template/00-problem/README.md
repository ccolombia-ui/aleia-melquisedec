# 00-problem/ - MELQUISEDEC (Definición del Problema)

Esta carpeta contiene la definición formal del problema que motiva esta épica de investigación.

## Estructura

```
00-problem/
├── problem-statement.md       # Problema central y pregunta de investigación
├── stakeholders.md            # Stakeholders y roles DAATH-ZEN
└── scope-and-constraints.md   # Alcance, límites y constraints
```

## Rostro DAATH-ZEN

**MELQUISEDEC** (Problema): Define el problema, valida requisitos, asegura alineación con objetivos.

## Checkpoint

**CK-01**: Validar que el problema está bien definido antes de avanzar a investigación.

## Criterios de Validación

- [ ] problem-statement.md completo con pregunta SMART
- [ ] Success criteria medibles y alineados
- [ ] Stakeholders identificados con responsabilidades
- [ ] Alcance y constraints documentados
