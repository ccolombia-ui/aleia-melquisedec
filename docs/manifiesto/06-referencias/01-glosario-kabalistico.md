# Glosario Kabalístico

Este glosario reúne términos usados en el manifiesto y en la organización MELQUISEDEC, con definiciones breves y referencias.

- **Tzimtzum**: Principio de constricción intencional; en MELQUISEDEC indica límites procesables en un flujo para evitar sobreproducción y respetar foco.

- **Rostro**: Actor organizacional o agente funcional (e.g., HYPATIA, SALOMON, ALMA) responsable de un conjunto de tareas.

- **Autopoiesis**: Capacidad del sistema para auto-producirse y mantener su identidad mediante ciclos de feedback.

- **Research Instance**: Unidad de trabajo completa (0-inbox..5-outputs) que documenta desde la issue hasta los outputs finales.

- **HKM**: Header Knowledge Management; metadata estándar obligatoria para artifacts.

- **SECI**: Modelo de trazabilidad (derives_from / informs) basado en Nonaka para mapear conocimiento explícito/tácito.

- **Output**: Entregable final, publicado y versionado, inmutable una vez publicado.

- **Lesson**: Aprendizaje documentado post-mortem con causa raíz y mitigación.

- **Inbox**: Punto de entrada universal para issues y tareas (`0-inbox/`).
