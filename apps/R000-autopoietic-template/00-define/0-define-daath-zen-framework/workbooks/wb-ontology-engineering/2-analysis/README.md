# 2-analysis/ - Critical Analysis

## Purpose

This folder contains critical analysis of themes, patterns, and insights discovered across sources in `1-literature/`. It synthesizes information from multiple sources to identify recurring concepts, controversies, and gaps.

## File Organization

- `themes-{topic}.md`: Thematic analysis of major topics
- `patterns-{aspect}.md`: Recurring patterns or practices
- `controversies-{debate}.md`: Debates or conflicting viewpoints
- `gaps-{area}.md`: Identified research gaps

## Quality Criteria

- **4-5 themes minimum** for comprehensive analysis
- **Multi-source themes** (each theme references 2-3+ sources)
- **Critical evaluation** (not just summary - analyze strengths/weaknesses)
- **Cross-references** to 1-literature/ sources
- **Comparative analysis** (similarities/differences between sources)

## Validation Checklist

- [ ] At least 4 theme analysis documents
- [ ] Each theme references 2-3+ sources from 1-literature/
- [ ] Comparative tables showing cross-source analysis
- [ ] Synthesis section with agreements/disagreements
- [ ] Critical evaluation of strengths/weaknesses
- [ ] Cross-references to related atomics (3-atomics/)
- [ ] Implications for target system (ALEIA-MELQUISEDEC)
